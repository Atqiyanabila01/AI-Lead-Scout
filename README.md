# 🔭 AI Lead Scout Agent

An automated research tool that crawls websites, summarizes content using AI logic, and generates professional business reports.

## ✨ Features
- **Asynchronous Crawling**: High-speed data collection via `Crawl4AI`.
- **Dual-Export**: Simultaneously updates `.csv` and formatted `.xlsx` files.
- **Auto-Formatting**: Automatically sets column widths and text wrapping in Excel.

## 🛠️ Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run the agent: `python lead_scout.py`.
